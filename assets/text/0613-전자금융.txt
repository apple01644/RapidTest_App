# 가입신청
- 서면으로 서비스 가입신청({본인,대리인;법정대리인})
 - 인터넷 뱅킹
 - 스마트 뱅킹(인터넷 뱅킹 가입 후 가능)
- 사용인감계 제출 시 1회로 1회 사용 가능
- 신설법인: 대표이사 개인인감증명서, 실명확인으로 신청

#이체한도
- 인터넷뱅킹
 - 기본 천만원
  - 보안카드: {개인;법인} 천만원 이하
  - OTP: 
   - 개인 1일:{1억},1일{5억} 이하
   - 법인 1일:{5억},1일{10억} 이하
- 폰뱅킹
 - 개인: 보안(5백만), OTP(500만/2500만) 
 - 법인 OTP(1억/5억)
- 한도 이내는 담당책임자 전결
- 한도 초과는 영업점장 전결
# 출금계좌

 - {보통,신자유저축,기업자유,당좌,가계당좌,외화보통예금,외화당좌예금;거치식;적립식}예금
  - 외화보통예금, 외화당좌예금: 동일인 요구불/당타행 외화예금 이체 가능
- 폰뱅킹(개인,법인,개인사업자)
 - 개인: {보통,신자유저축,기업자유,당좌,가계당좌;거치식;적립식,외화보통예금,외화당좌예금}예금
 - 기업: {보통,기업자유;거치식;적립식,외화보통예금,외화당좌예금}예금
- 법인/단체의 유통장 계좌로 지정할 시: 통장 실물 {필요;불요}
 - 공동명의는 지정불가
#보안매체
- 기업뱅킹은 {일반OTP,카드OTP;디지털OTP;스마트OTP;보안카드}만 발급 가능
- 장애인용 보안매체 발급 시 시각장애인 확인서류 징구
- 보안매체는 고객당 1개 발급
 - 기업뱅킹/법인폰뱅킹은 이용자ID/번호 별로 1매씩 발급 가능
- 발급수수료 면제
 - 보안카드: {에이스;톱;골드;로얄} 등급 이상 또는
  - 일반보안카드 {발급;재발급}, 점자보안카드 발급/재발급
 - OTP: {톱;골드;로얄;에이스} 등급 이상
   - 카드 OTP: {골드;로얄;에이스;톱} 등급 이상 또는 주무부서장 승인
  - {2}년이내 기기불량/배터리소진
  - 보이스OTP 발급/재발급
  - 디저털OTP 발급/재발급
 
- OTP 유효기간 2년

# 예약이체
- 3회에 걸쳐 이체 시도
 - 10시, 14시, 16시 반
# 지연이체
 - 3~5시간 지연 가능
  - 이체 예정시간 30분내 취소 가능
 - 지연예외처리
  - 예외이체한도 이내 금액
  - 예외입금계좌
  - 본인 명의계좌 입금
  - 당행 가상계좌, 퇴직연금계좌
# 보안계좌
 - 대상: 요구불,예적금,신탁,수익증권,외화예금
  - BC카드 결제계좌 불가
 - 제한: 보안계좌는 전자금융 출금계좌로 등록 불가
# 안전계좌조회
 - 실제계좌번호와 예금주명을 검열 후 거래내역, 잔액 공개
# 자산분리계좌
 - 해당 서비스 신청시 자산분리계좌만 전가금융거래에 조회 됨
# 입금계좌 지정
 - 지정된 계좌에만 입금 가능
 - 미지정계좌에 대한 입금허용 금액: 0~{1백만}원 설정 가능
  - 서면을 통해 가능
# 휴대폰 문자통지
 - 입출금 통지
  - 정액제 월 1000원
  - 종량제 건당 30원
 - 주요환율통지
  - 통화별 월 1500원
 - 펀드평가액, 수익률 통지
  - 계좌별 월 1000원
 - 신청일 익월부터 매 10일에 출금
 - 3개월 미납 시 자동해지
 - 미납수수료 있는 계좌로 {1}년 이내 재가입 시 수수료 장구 
# 공동인증서
 - 수수료 환급: 발급 후 7일이내 가능
  - 세금계산서 신청 기록 有 고객 {제외;포함}
 - 추가인증
  - 1일 누적 100만우너 이상 자금이체 시 추가인증
   - {OTP;보안카드} 생략
